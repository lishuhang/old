图集：和 Windows XP 一起走过的日子

发表评论

Windows XP操作系统到今天就正式地停止技术支持了。但是随着过去一段时间，它寿终正寝的倒计时越来越近，我却不禁把目光更多地注意到这个系统身上。因为我发现，在过去的日子里，它无论何时，总与我相依相伴，伴我成长，在各种意想不到的地方出现。

也许平时，在一切都运作正常的时候，我们不一定注意得到日常生活当中它们默默无言坚守岗位的身影，也许它们一直是我们生活场景的背景或者点缀；只有当它们累了，显示出一张蓝色的病颜，需要休息一下的时候，我们才能瞥见其真容。

这个图集就是我在过去的5-6年当中拍摄到的，一些关于Windows XP支持我们日常生活的身影。可以想见，这些机器和其上的XP系统，仍将在以后的很多年里继续服役，一直到它们当中最后一台机器完全停止转动为止。
